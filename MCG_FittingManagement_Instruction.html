MCG Fitting Management

Công cụ bóc tách hình học, quản lý catalog fitting theo dự án, xuất BOM và đánh balloon cho bản vẽ AutoCAD.

Lệnh gọi: MCG_Fitting Tab chính: Fitting Table .NET 4.8 · WPF

Giới thiệu

MCG Fitting Management hỗ trợ toàn bộ vòng đời quản lý fitting của một dự án. Mỗi dự án tương ứng 1 Project Folder chứa file catalog FittingCatalog.json và các block .dwg. Trung tâm của plugin là tab Fitting Table — nơi import, quản lý, sao chép và chèn fitting.

📋 Fitting Table TAB CHÍNH

Import từ Inventor, thêm từ CAD, quản lý catalog theo cây Category, sao chép fitting sang dự án khác, chèn bảng tổng hợp.

📊 Fitting Handle

Quét bản vẽ, đếm fitting, gán số Position, đồng bộ vào CAD, xuất ma trận BOM ra Excel và đánh balloon.

🧱 Block Utilities

Đổi tên, định nghĩa lại, thay thế, đổi điểm chèn, thêm/tách đối tượng khỏi block, gom nhiều bản vẽ.

1. Khởi động plugin

  1. Gõ lệnh MCG_Fitting vào dòng lệnh AutoCAD rồi nhấn Enter.
  2. Bảng công cụ (PaletteSet) “Fitting Management” hiện ra — có thể neo (dock) sang trái/phải màn hình.
  3. Tab Fitting Table nằm ở vị trí đầu tiên — bắt đầu làm việc từ đây.
MẹoPalette ghi nhớ vị trí và thứ tự tab theo phiên làm việc. Sau khi bấm một nút trên Palette, focus tự trả về bản vẽ để bạn pick điểm / chọn đối tượng ngay.

⭐ 3. Tab chính — Fitting Table

Tab Fitting Table chỉ có 1 nút Open Fitting Table để mở cửa sổ Fitting Table — nơi tập trung toàn bộ thao tác quản lý fitting của Project Folder đang mở. Tất cả các mục 3.1 → 3.10 dưới đây đều nằm trong cửa sổ này.

3.1 — Mở cửa sổ & chọn Project Folder

  1. Ở tab Fitting Table, bấm Open Fitting Table.
  2. Góc trên phải cửa sổ: ô Project hiển thị dự án đang mở. Bấm Load Folder để chọn Project Folder.
  3. Chọn (hoặc tạo mới) thư mục dự án.

3.2 — Bố cục cửa sổ

Khu vựcChức năng
Thanh trênTiêu đề FITTING TABLE; bên phải là ô Project + nút Load Folder.
Global SearchTìm đa từ khóa theo Part ID, Name, Description, Originator, View — gõ nhiều từ cách nhau bằng dấu cách (ví dụ sheave 22).
Cột trái — CategoriesCây nhóm fitting theo Name. Kéo-thả sắp xếp, multi-select (Ctrl/Shift), chuột phải thao tác cả nhóm (xem 3.5, 3.7).
Giữa — GridBảng fitting: Type, Source, View/File, Part Complete ID, Name, Description, Project Pos. (ô vàng), UoM, Weight, Originator, Created + các cột động (xem 3.8).
Phải — PreviewXem trước hình + toàn bộ thuộc tính (kể cả ExtraProperties động từ iProperty/Vault).

3.3 — Add from Inventor (import .idw)

Import fitting từ file Inventor Drawing được thực hiện ngay trong cửa sổ Fitting Table (nút ➕ Add from Inventor ở góc dưới trái, cạnh Add from CAD).

  1. Bảo đảm đã Load Folder (có Project đang mở). Nếu chưa, plugin báo “Load a folder first”.
  2. Bấm ➕ Add from Inventor.
  3. Hộp thoại Add from Inventor hiện ra — chọn Target BOM Type: Equipment (mặc định) hoặc Hull, rồi bấm Select .idw….
  4. Chọn 1 hoặc nhiều file .idw. Plugin bóc tách hình học/thuộc tính, tách mỗi view thành block, ánh xạ layer, bơm attribute và ghi vào catalog của dự án đang mở.
  5. Kết thúc hiện hộp thoại kết quả (số thành công/thất bại + trạng thái Vault). Bảng tự làm mới.

3.4 — Add from CAD

Thêm fitting vào catalog bằng cách chọn đối tượng trong bản vẽ:

  1. Bấm ➕ Add from CAD — cửa sổ tạm ẩn để bạn thao tác trên bản vẽ.
  2. Pick đối tượng/feature hình học theo hướng dẫn ở dòng lệnh.
  3. Điền thông tin trong hộp thoại Virtual Item rồi lưu — fitting được thêm vào catalog.

3.5 — Menu chuột phải (trên 1 dòng fitting)

LệnhTác dụng
Edit PropertiesSửa thuộc tính fitting (không áp dụng cho item nguồn Inventor).
Edit Pos NumSửa Project Pos Num của fitting.
Edit View TypeĐánh dấu Plan View / quy tắc đếm Qty theo Plan View cho Hull BOM.
SyncCập nhật AttributeDefinitions của block và mọi INSERT trong bản vẽ hiện hành.
Sub-BOM / AccessoriesQuản lý phụ kiện (accessories) gắn kèm fitting.
Copy to…Sao chép fitting sang một Project Folder khác — xem 3.6.
Remove from ProjectGỡ vĩnh viễn fitting khỏi dự án hiện tại.

Menu này cũng xuất hiện khi chuột phải trên node Category (áp dụng cho toàn bộ fitting trong nhóm).

3.6 — Copy to… (sao chép sang dự án khác)

Cho phép sao chép fitting đang chọn sang một Project Folder khác và tự cập nhật FittingCatalog.json của dự án đích — không làm thay đổi dự án đang mở.

  1. Chọn 1 hoặc nhiều fitting trong Grid (giữ Ctrl/Shift để chọn nhiều).
  2. Chuột phải → Copy to…
  3. Chọn Project Folder đích (phải khác dự án hiện tại).
  4. Plugin sẽ: sao chép file .dwg của block sang folder đích, nhân bản bản ghi catalog (kèm Accessories), trỏ đường dẫn về bản copy, rồi cập nhật FittingCatalog.json của dự án đích.
  5. Thông báo tổng kết: số item đã copy + số file .dwg đã sao chép.
Phạm vi sao chépBao gồm bản ghi catalog + file .dwg của block + toàn bộ Accessories / Sub-BOM gắn kèm fitting. Dự án đích dùng được độc lập, không phụ thuộc dự án nguồn.

3.7 — Cây Category (sắp xếp & thao tác nhóm)

  • Kéo-thả các Sub Folder (Level 2 — mỗi Sub Folder = 1 Part Number) để đổi thứ tự hiển thị. Việc này chỉ đổi thứ tự hiển thị, không ghi đè Project Pos Num nghiệp vụ.
  • Multi-select: Ctrl-click để chọn/bỏ từng node; Shift-click để chọn dải liên tục (cùng nhóm cha).
  • Chuột phải trên Category → áp dụng thao tác (Edit/Sync/Copy to/Remove…) cho toàn bộ fitting trong nhóm đang chọn.

3.8 — Tùy biến cột (Customize Grid)

Chuột phải lên tiêu đề cột → menu Customize Grid: tích/bỏ tích để bật/tắt bất kỳ cột nào. Cột cố định mặc định hiện; cột động (iProperty/Vault field) mặc định ẩn. Lựa chọn được lưu lại và dùng chung cho mọi dự án.

3.9 — Thanh nút dưới cùng

➕ Add from CAD
Thêm fitting từ đối tượng trong bản vẽ (3.4).
➕ Add from Inventor
Import fitting từ file .idw vào dự án đang mở (3.3).
Auto-Assign Pos
Tự gán số Position hàng loạt cho fitting DETAIL/HULL — hộp thoại cho chọn số bắt đầu (nhập 50 → 050, 051…).
Push Update
Ghi đè file .dwg trong Project Folder bằng định nghĩa block từ bản vẽ đang mở.
Refresh
Nạp lại dữ liệu catalog từ file.
Insert Fitting Table
Chèn bảng lưới tất cả fitting (hoặc Category đang chọn) vào bản vẽ (3.10).
Insert to CAD
Chèn (các) fitting đang chọn vào bản vẽ dưới dạng block. Double-click 1 dòng cũng chèn.

3.10 — Chèn bảng Fitting Table vào CAD

Nút Insert Fitting Table vẽ 1 bảng tổng hợp trực tiếp trong Model Space: mỗi hàng 1 fitting, gồm Pos / Part Number / Title / Views / Description / Material / Mass / UoM.

  • Tiêu đề bảng = tên Category đang chọn (chữ trắng, cỡ lớn); dòng “Created: yyyy-MM-dd HH:mm” ở cuối bảng. Không chọn category cụ thể (“All Fittings”) → tiêu đề là tên Project Folder.
  • Cập nhật 1-click: click vào bảng cũ đã chèn → tự vẽ lại đúng vị trí cũ, không hỏi lại điểm chèn; ngày “Created” giữ nguyên.
  • Tô đỏ thay đổi: khi cập nhật, hàng có dữ liệu thay đổi so với lần trước hiện chữ đỏ.

4. Tab — Fitting Handle

4.1 — Xuất BOM (BOM Export)

Nhóm “BOM EXPORT” có 2 nút mở cửa sổ BOM Matrix Export & Sync:

Open Equipment BOM
BOM cho Thiết bị (Equipment). Đếm fitting theo từng Equipment.
Open Hull BOM
BOM cho phần Vỏ (Hull). Hỗ trợ đếm theo Plan View khi 1 fitting có nhiều mặt cắt trong cùng khung A1.

Quy trình chuẩn trong cửa sổ BOM Matrix:

Scan Drawing
Quét bản vẽ, nhận diện block fitting và dựng ma trận BOM.
Auto-Assign Positions
Tự gán số Position cho fitting chưa có số.
Sync Pos to CAD
Ghi ngược số Position vào attribute của block trong bản vẽ.
Export to Excel
Xuất ma trận BOM ra file Excel (.xlsx).

Chuột phải trên bảng có “Sync Pos from Item Library”: đọc lại số Position từ catalog dự án và làm mới bảng.

Lưu ý thứ tựLuôn Scan → Auto-Assign Positions → Sync Pos to CAD → Export to Excel. Export trước khi Sync thì số Position trong bản vẽ chưa được cập nhật.

4.2 — Đánh Balloon (Ballooning)

Place Balloon
Đặt balloon thủ công. Click liên tục lên từng fitting; nhấn Esc để kết thúc.
Mass Auto-Balloon
Đánh balloon hàng loạt theo vùng. Chọn đường bao → tự gắn balloon cho mọi fitting trong vùng.

5. Tab — Block Utilities

5.1 — Công cụ Block

Chỉnh sửa block đã có mà không phá vỡ tham chiếu. Mỗi nút yêu cầu chọn đối tượng trong bản vẽ:

NútChức năng
Rename BlockĐổi tên (nhân bản) một block.
Redefine BlocksĐịnh nghĩa lại block từ bản vẽ đang mở.
Replace BlockThay block này bằng block khác (giữ vị trí/scale).
Change Insertion PointĐổi điểm chèn (base point) của block.
Add Objects to BlockThêm đối tượng vào định nghĩa block đang có.
Extract Objects from BlockTách đối tượng lồng bên trong ra khỏi block.

Quy trình làm việc đề xuất

  1. Mở dự án: Tab Fitting Table → Open Fitting Table → Load Folder chọn Project Folder.
  2. Xây dựng catalog: Add from Inventor (chọn Equipment/Hull → .idw) và/hoặc Add from CAD.
  3. Quản lý & gán Pos: sắp xếp Category, Auto-Assign Pos, Push Update khi cần.
  4. Tái sử dụng: chuột phải Copy to… để chuyển fitting sang dự án khác.
  5. Chèn vào bản vẽ: Insert to CAD (từng fitting) hoặc Insert Fitting Table (bảng tổng hợp).
  6. Lập BOM & Balloon: Tab Fitting Handle → BOM (Scan → Auto-Assign → Sync → Export) và Place/Mass Auto-Balloon.
